# Finance tracker PWA

This is a simple private finance tracker.

## What it does

- Tracks manually entered accounts, investments and spending
- Shows balance totals and simple forecasts
- Works offline once installed
- Stores data on your device using browser local storage
- Lets you export and import JSON backups

## How to use it

For proper installation on Android, upload the whole folder to a web host such as GitHub Pages, Netlify or Vercel.

Then open the hosted URL in Chrome and choose:

Menu > Add to Home screen

or tap the in-app Install button if Chrome displays it.

## Important

This is not connected to your bank. It is a manual tracker and does not provide financial advice.
